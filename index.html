Airline Hubs Webapp

The impending pilot shortage gives new pilots more options on which airline they would like to work for. As pilots gain seniority in an airline, they have the opportunity to choose which hub they would like to be based out of. It is not advantageous to hop between airlines, so a pilot is usually stuck working for the same airline until they retire. Since there is no consolidated information or visual of each airline’s hub, a pilot cannot easily rank which airline is best for them based off of hub locations. I am planning on pulling together the hubs for each major airline, the large cargo airlines, and larger regional airlines and map them all in one place. My initial thought is to use CartoDB because of the robust information I can gather on airlines and store in a database. There is no consistent information and the data gathering, validation, and consolidation will be the biggest part of this project. From there I can play around with the data visualization tools CartoDB has to offer.

South Philly Open Data

The main idea for this project is to use Philadelphia’s open data to create a trash bin desert map in South Philadelphia. While recently walking through South Philly, I noticed that there was an abundance of trash everywhere. There were also almost no trash cans to be found. The residents of South Philly can’t be expected to throw their trash away if there is nowhere to do so. I thought the best way would be to place trash bins in the locations that are located in the middles of so-called “trash bin deserts.” The city of Philadelphia’s open data portal has the data for the location of their waste wire trash bins as well as the Big Belly trash bins.

Current status: data gathering

Web App - Bing Maps

This was my first application created using JavaScript. It created in the GIS Application Development class in my master's program using Eclipse IDE and Bing Maps AJAX controls. While the application looks a little anachronistic, it was fun building an application that incorporates maps and learning how tinker with Bing Maps API and JavaScript in general.

The application is a map of the Washington, D.C. area and all of the Smithsonian Museums in the district. It includes a list of the museums with a link to their respective pages. Info boxes pop up when you mouse over the lists or custom icons and disappear on mouse out. The lists and the picture in the info box link to the museum webpage. Also included is a "Get Directions" button that routes out the directions to three different museums.
